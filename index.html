El Problema de la Signatura Lorentziana

¿De dónde sale (-,+,+,+)?

La signatura del espacio-tiempo no es un "detalle técnico" - define toda la estructura causal:

  • Conos de luz: Qué eventos pueden influenciarse
  • Tiempo vs Espacio: Distinción fundamental
  • Causalidad: Pasado no puede afectar su propio pasado

Opciones en LQG:

1. SU(2) - Formulación Euclidiana

Métrica: ds² = dx² + dy² + dz²

Signatura: (+,+,+)

✗ Solo espacio, falta el tiempo. Necesitas "rotar" analíticamente t → it (continuación de Wick)

2. SL(2,ℂ) - Formulación Lorentziana

Métrica: ds² = -dt² + dx² + dy² + dz²

Signatura: (-,+,+,+)

✗ Representaciones infinito-dimensionales, no unitarias, espectro complejo

Trampa Conceptual:

La elección del grupo gauge ya codifica la signatura.

  • Selección dinámica: Solo configs lorentzianas son estables
  • Condensado cuántico: Fase tipo BEC con simetría emergente

🛠 Soluciones Propuestas y Progreso

1. Estados Coherentes Mejorados

Usar superposiciones gaussianas sobre grafos regulares para aproximar métricas clásicas. En la simulación, esto se ve al aumentar densidad.

|ψ⟩ = ∫ Dg exp(i S[g]/ℏ) |g⟩

Aproximación saddle-point para límite clásico.

2. Emergencia de Signatura

  • Condensado geométrico: Análogo a BEC donde signatura emerge de fase colectiva (ver GFT).
  • Group Field Theory (GFT): Campos sobre grupo de Lorentz, emergencia via condensación de "átomos de espacio".
  • Renormalización: Flujo RG muestra transición de fase a régimen lorentziano, resolviendo circularidad.

3. Dinámica Efectiva

Modelos efectivos como LQC resuelven singularidades (big bounce), mostrando emergencia de GR efectiva.

ρ_efectiva = ρ (1 - ρ/ρ_crit)

Donde ρ_crit ~ 1/ℓ_p³, evitando big bang singular.

4. Espinfoams con Twistores

Incorporar twistores para codificar estructura causal sin foliación previa, permitiendo emergencia real.

Progreso Reciente (2020s):

  • Simulaciones numéricas de espinfoams muestran convergencia a GR en límites grandes (e.g., trabajos de Dittrich).
  • Conexiones con tensor networks para emergencia holográfica de causalidad.
  • Propuestas para tests via cosmología cuántica (e.g., CMB anomalies).
  • Formulaciones covariantes sin gauge fixing, reduciendo circularidad.

Estos enfoques no resuelven todo, pero muestran caminos viables. La simulación ilustra visualmente esta transición.
